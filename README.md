# Violet Void Theme for Atuin

A dark purple-tinted theme for [Atuin](https://atuin.sh/) shell history.

## Install

```bash
# Copy theme to Atuin config
mkdir -p ~/.config/atuin/themes
cp violet-void-theme_atuin.toml ~/.config/atuin/themes/

# Add to your atuin.toml
theme = "violet-void-theme"
```
